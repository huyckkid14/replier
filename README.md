# Replier Coming Soon Website

Static coming-soon website for Replier, an AI email assistant Chrome extension for Gmail.

## Pages

- `index.html` - public coming-soon landing page.
- `privacy.html` - Replier privacy policy, last updated June 25, 2026.

## Local Preview

Open `index.html` directly in a browser, or serve the folder:

```sh
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deploy

This repository has no build step. Deploy the repository root as a static site with GitHub Pages, Cloudflare Pages, Netlify, Vercel, or any static web host.
